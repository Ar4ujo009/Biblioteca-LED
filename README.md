
# Biblioteca LED

Biblioteca simples para controle de LED de forma mais simplificada dentro do Arduino e ESP32.

A classe 'Led' permite:

- ligar o LED
- ligar por tempo determinado
- apagar
- altenar estado
- piscar continuamente
- piscar uma quantidade definidamente de vezes

O funcinamento é não bloqueante, usando "millis()".
Por isso, o mêtodo "update" deve ser chamado repetidamente dentro do "loop()"
